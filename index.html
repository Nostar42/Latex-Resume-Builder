Quick Start

  1. Install MiKTeX — download from miktex.org/download. Choose Install for all users and enable Install missing packages on-the-fly.
  2. Install Ollama and pull a model 
    winget install Ollama.Ollama
ollama pull qwen2.5-coder:3b   # recommended starting point
  3. Clone the repository 
    git clone https://github.com/Nostar42/Latex-Resume-Builder.git
cd Latex-Resume-Builder
  4. Run one-time setup (installs Python, pip packages, nginx) 
    .\setup.ps1
  5. Launch — double-click start.bat, or: 
    .\start.ps1
    Then open http://localhost:5000 in your browser.

Environment Variables

$env:OLLAMA_MODEL     = "qwen2.5-coder:7b"   # default model
$env:OLLAMA_URL       = "http://localhost:11434"
$env:SESSION_MAX_AGE  = "86400"              # session TTL in seconds

# Switch to vLLM (GPU server):
$env:USE_VLLM         = "true"
$env:LLM_BASE_URL     = "http://localhost:8001"

Troubleshooting

"pdflatex not found" — install MiKTeX and verify with pdflatex --version in PowerShell.

First compile is slow or fails — MiKTeX downloads missing packages on first use (60–120 s). Just try again; subsequent compiles are fast.

"Ollama isn't running" — run ollama serve in a terminal, or check the Ollama tray icon.

"Can't reach the model" — run ollama list; if empty, pull a model first.

Port 5000 in use — start.ps1 frees it automatically, or run:

Get-NetTCPConnection -LocalPort 5000 -State Listen |
  ForEach-Object { Stop-Process -Id $_.OwningProcess -Force }

setup.ps1 won't run — bypass policy for this one script:

powershell -ExecutionPolicy Bypass -File setup.ps1

Security Notes

-no-shell-escape is passed to every pdflatex call — LaTeX source cannot execute shell commands.

Session IDs are validated as UUID4 before use as filesystem paths (prevents path traversal).

Rate limiting is enforced at both nginx and FastAPI layers. Designed for personal/LAN use.
